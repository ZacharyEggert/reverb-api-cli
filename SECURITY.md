# Security Policy

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, email security@reverbdotcom.com with:

- A description of the vulnerability
- Steps to reproduce
- Potential impact

You should receive a response within 48 hours. If you don't, follow up to ensure the original message was received.

## Supported Versions

| Version | Supported |
|---------|-----------|
| latest  | Yes       |
